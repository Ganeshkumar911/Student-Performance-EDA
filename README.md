# Student Performance Analysis - Exploratory Data Analysis

## Project Objective

The objective of this project is to analyze student performance data and identify factors affecting academic success.

## Questions Explored

- Does study time affect marks?
- Does attendance influence performance?
- Which subject has the highest average score?
- Are there any outliers?
- What factors correlate with academic success?

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

## Visualizations

1. Study Hours vs Average Score
2. Attendance vs Average Score
3. Subject Wise Average Scores
4. Correlation Heatmap
5. Score Distribution

## Key Findings

- Higher study hours generally improve scores.
- Better attendance is associated with better performance.
- Strong correlations exist between academic variables.
